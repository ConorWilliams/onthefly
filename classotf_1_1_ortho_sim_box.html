otf::OrthoSimBox class
#include <libatom/system/simbox.hpp>

Provides details of the simulations Orthoganal supercell geometry,.

All queries of the space in which the atoms exist are provided by this class. It is assumed (and must be ensured) all non-periodic atoms are within the OrthoSimBox extents.

template<typename T>
auto canon_image(Eigen::ArrayBase<T> const& x) const -> Vec<flt_t> noexcept
Maps atom into canonical cell, 0 <= r_i < extent_i for all i which are periodic. Non-periodic atoms are within the simbox extents so x[i] * inv_extents less than 1 and x[i] remains unaffected, hence no non-periodic switch/select.
template<typename A, typename B>
auto min_image(Eigen::ArrayBase<A> const& a, Eigen::ArrayBase<B> const& b) const -> Vec<flt_t> noexcept
Compute the shortest vector connecting a to a periodic image of b. This function is branchy and should be avoided in hot code.
